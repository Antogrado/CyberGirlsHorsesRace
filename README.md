# CyberGirlsHorsesRace

Автор: Милиса/Мельханов, 25ИС11д

## Описание
Киберскачки (Веб-приложение для виртуальных ставок на скачки с элементами игры)

## План функций
- Начать забег
- Показать изменения движений лошадей в забеге
- Определить победившую лощадь
- Завершить забег
- Показать результаты
  
## Стек
C# / ASP.NET Core Web api / Postgre SQL / Visual Studio

## Как запустить
1) Открыть solution в Visual Studio
2) Нажать F5 (Run)
