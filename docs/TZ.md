# Мини-ТЗ: C.G.H.R gg™

## 1. Название
Киберскачки (Веб-приложение для виртуальных ставок на скачки с элементами игры)

## 2. Цель
Развлекательное приложение, симулирующее забеги виртуальных лошадей, с системой ставок, характеристиками «скакунов» и динамическим отображением гонки

## 3. Пользователь
Обычный пользователь ПК

## 4. Функциональные требования (FR)
- FR1: Запуск и визуализация гонки. Динамическое отображение позиций лошадей (полоса анимации сверху и фишки на трассе).
- FR2: Выбор режима («Своя конюшня» ставка на любую лошадь / «Случайные лошади» твоя лошадь против ботов).
- FR3: Система ставок. Возможность сделать ставку на выбранную лошадь, просмотр ее характеристик (скорость, выносливость и т.д.).
- FR4: Лог событий. Автоматическое ведение текстовой трансляции забега с юмористическими комментариями

## 5. Нефункциональные требования (NFR)
- Приложение стабильно работает в современных браузерах.
- Интерфейс интуитивно понятен (три колонки: управление, трасса, лог).
- Анимации гонки и обновление лога происходят плавно, без «зависаний».

## 6. Модель данных (черновик)
**Лошадь (Horse)**
- Id (int)
- Name (string)
- Number (int)
- Speed (int)
- Stamina (int)
- Luck (int)
- 
**Забег (Race)**
- Id (int)
- Status (string) 
- Result (json)
- 
**Ставка (Bet)**
- Id (int)
- RaceId (int)
- HorseId (int)
- Amount (decimal)

## 7. Критерии приемки (как проверить)
- Можно запустить новый забег, выбрав лошадей.
- Можно сделать ставку на любую лошадь в предстартовом меню.
- Во время гонки в реальном времени отображается движение лошадей и пополняется лог событий.
- По окончании забега баланс пользователя корректно увеличивается/уменьшается в зависимости от результата ставки.
- Лог событий отображает ключевые моменты гонки.
